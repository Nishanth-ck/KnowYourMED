HOW TO RUN IN SOME TIME IN FUTURE :

1. Need a .env file with these parameters : 
PORT ----- the port no
MONGO_URL ----- the mongodb atlas url
SEC_ACC ----- some random string
SEC_REF ----- some random string
EMAIL_PASS ----- generated for a gmail account used here in place of 'knowyourmed1@gmail.com' to 'webdevelepor001nishanth@gmail.com' 
(go to google account -> security -> 2 step verification -> app passwords in search bar -> 'nodemailer' -> get the pass key )

2. to run : use two terminals :
    cd frontend -> npm install --legacy-peer-deps -> npm run dev -> it will start on : http://localhost:5173/
    cd backend -> npm install -> npm start -> it will start on : http://localhost:3000/ (the port number)

(special care : generated qr contains http://localhost:5173/ url)
